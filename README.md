# Landic Design System

## Installation

1. Clone this repository
2. `npm install`

## Usage

### Development server

```
npm start
```

Development server located at `localhost:8000`.

### Build

```
npm run build
```

Build output directory: `/dist`

### Compile CSS

For compiled bundle:

```
npm run sass
```

For minified bundle:

```
npm run sass-min
```

CSS output directory: `/css`
